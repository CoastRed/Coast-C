# 一维数组

## 数组的创建

语法：数组元素类型[] 数组的名字;

## 数组的初始化

```CSharp
int[] a = new int[3];
a[0] = 1;
a[1] = 2;
a[2] = 3;

int[] b = new int[] { 1, 2, 3 };

int[] c = { 1, 2, 3 };
```

# 二维数组

## 数组的创建

语法：数据类型[,] 数组名;
          数据类型[][] 数组名;

```CSharp
int[,] d = new int[2, 4];

int[][] e = new int[2][];
e[0] = new int[2];
e[1] = new int[3];
```

## 数组的初始化

```CSharp
int[,] f = new int[2, 2];
f[0, 0] = 1;
f[0, 1] = 2;
f[1, 0] = 3;
f[1, 1] = 4;

int[][] g = new int [2][];
g[0] = new int[] { 0, 1 };
g[1] = new int[] { 2, 3 ,4 };

int[,] h = new int[2, 2] { { 0, 1 }, { 2, 3 } };
int[,] l = new int[,] { { 0, 1 }, { 2, 3, } };
int[,] m = { { 0, 1 }, { 2, 3 } };
```
